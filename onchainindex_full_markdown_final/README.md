# OnChainIndex

OnChainIndex is an independent index that ranks on-chain applications by real usage.

No ads. No reviews. No paid placement.  
Just publicly verifiable on-chain data.

## What we measure
- Unique active wallets
- Transaction activity
- Growth trends over defined time windows

## What rankings are not
Rankings reflect usage only and are not endorsements, investment advice, or safety guarantees.

## Transparency
All rankings are calculated daily and preserved as historical snapshots.
Our methodology is public and versioned.
