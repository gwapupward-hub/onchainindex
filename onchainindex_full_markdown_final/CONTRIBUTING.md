# Contributing to OnChainIndex

Thanks for your interest in contributing.

OnChainIndex prioritizes correctness, neutrality, and transparency.
All contributions should align with these principles.

## What we welcome
- Bug fixes
- Performance improvements
- Documentation clarity
- Tooling and infrastructure enhancements

## What we donâ€™t accept
- Changes that affect rankings without methodology updates
- Promotional content or sponsored logic
- Opinionated scoring criteria

## Process
1. Open an issue describing the change
2. Discuss before submitting a PR
3. Keep changes focused and minimal

## Code standards
- Deterministic logic
- Clear naming
- No hidden heuristics

## Governance
OnChainIndex is maintained with editorial responsibility.
Final decisions rest with the maintainers to preserve neutrality.
