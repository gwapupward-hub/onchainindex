# Press

## About OnChainIndex

OnChainIndex is an independent index that tracks and ranks on-chain applications based on real usage.

Rankings are calculated using publicly available blockchain data, including metrics such as unique active wallets and transaction activity over defined time windows. Rankings are updated daily and preserved as historical snapshots.

OnChainIndex does not accept paid rankings, reviews, or self-reported metrics. Rankings reflect usage only and are not endorsements.

---

## What OnChainIndex Is Not

OnChainIndex is intentionally limited in scope.

- It is **not** an endorsement platform  
- It is **not** an audit, rating agency, or security review  
- It is **not** investment or financial advice  
- It does **not** measure code quality, safety, or legitimacy  
- It does **not** accept paid placements or promotional rankings  

Rankings reflect **usage only**, based on publicly available on-chain activity.

## Methodology Summary

- Rankings are calculated using on-chain activity data
- Primary metrics include unique active wallets and transaction counts
- Data is aggregated daily and stored as immutable historical snapshots
- Anti-spam measures are applied to reduce automated activity

The full methodology is publicly available and versioned at `/methodology`.

## Verification & Badges

### Verified by GwapScore
“Verified by GwapScore” indicates that a dApp meets baseline credibility signals based on on-chain activity. Verification does not affect rankings.

### Ranked on OnChainIndex
“Ranked on OnChainIndex” is used for attribution in widgets and embeds and does not imply endorsement.

## API & Integrations

OnChainIndex provides a public, read-only API and embeddable widgets for accessing rankings and historical data. All integrations require attribution.
